# 🚗 SMARTDRIVE

Sistema de controle de velocidade em estacionamentos usando **Arduino + radar + câmera** e app mobile (**React Native + Expo**).

---

## ⚙️ Funcionamento
- Mede velocidade do veículo  
- Se > 20 km/h:
  - Captura placa  
  - Exibe alerta  
  - Registra no sistema  

---

## 🧰 Tecnologias
Arduino • ESP32-CAM • Radar Doppler • React Native • Expo

---

## 🎯 Objetivo
Reduzir excesso de velocidade de forma **educativa e visível**.
